# 首个安卓预览版 / First Android Preview

这是 `Cultivation World Simulator Android` 的首个公开安卓预览版。

本项目基于上游仓库 `4thfever/cultivation-world-simulator` 修改而来，当前版本的目标是先把游戏主体跑通到安卓设备上，并开放一版可安装、可测试、可反馈的问题收集版本。

## 这版能做什么

- 安卓应用可以启动内嵌运行时并进入游戏
- 已完成第一轮移动端界面适配
- 安卓真机存档失败的核心链路已修复
- 可以从源码构建 `debug APK`

## 附件说明

- `app-debug.apk`
  - 可直接安装的调试包
  - 当前最适合普通测试者下载安装体验
- `app-release-unsigned.apk`
  - 未签名的 release 构建产物
  - 不适合作为普通玩家直接安装包
- `app-release.aab`
  - Android App Bundle 构建产物
  - 更适合作为后续正式发布链路的中间产物

构建时间：`2026-04-09`

当前阶段依然不属于正式稳定版发布，更适合作为首个公开安卓预览版。

## 安装说明

- 支持系统：Android 7.0 及以上（`minSdk 24`）
- 首次安装前，需要允许未知来源应用安装
- 如果你之前已经装过早期测试包，建议先卸载旧版本再安装这一版
- 首次启动时间可能较长，因为应用需要解包运行资源并初始化本地服务

## 已知限制

- 当前最适合直接安装体验的仍然是 `debug APK`
- `release APK` 还不是正式签名发布包
- 还没有完成正式签名与稳定发布流程
- 主要在 Windows 构建链路下完成验证
- 移动端界面仍在持续优化，不保证所有页面都已经完全适配手机屏幕
- 后台保活和长时间挂机能力目前不保证稳定
- 当前更适合当作预览版、测试版使用，不建议当作长期稳定版理解

## 重点反馈项

如果你准备反馈问题，建议至少带上这些信息：

- 机型
- 安卓版本
- 真机还是模拟器
- 问题出现时的操作步骤

当前最希望收集的反馈包括：

- 存档失败
- 启动卡住或闪退
- 页面挤压、遮挡、显示不全
- 事件栏或弹窗无法正常滚动
